The project is primarily developed using JavaScript with the Express framework for building the server-side application. It utilizes Mongoose as an ODM (Object Data Modeling) library to interact with a MongoDB database.

Key dependencies include:
- `express`: Web framework for Node.js.
- `mongoose`: ODM for MongoDB.
- `cors`: Middleware to enable CORS.
- `dotenv`: Module to load environment variables from a `.env` file.
- `ejs`: Template engine for rendering HTML.
- `jsonwebtoken`: Library for handling JWT (JSON Web Tokens).
- `cookie-parser`: Middleware to parse cookies.
- `short-id`: Library to generate short unique IDs.

The estimated entry point command to start the application is `npm run dev` for development or `npm start` for production.

Specific environment requirements:
- A `.env` file must be present with the following variables:
  - `PORT`: Default value is 8000.
  - `MONGO_URI`: MongoDB connection string.
  - `SECRET_KEY`: Secret key for JWT signing.
